这是一款组件包，提供了一些web开发常用的组件，而且这里的大部分组件都是可以复用的。参与开发可以直接向该包提交代码，将会同步至相应的包。

## 主要组件

- 基于 Psr7 的 max/http-message
- 基于 Psr11 的 max/container
- 基于 Psr14 的 max/event
- 基于 Psr15 的 http-Server
- 基于 Psr16 的 max/cache，支持 File,Memcached,Redis,APC [可扩展]
- 符合 Psr7 规范的 max/routing 路由组件
- 数据库 max/database 组件，支持连接池
- Session max/session
- Blade视图组件 max/view
- 验证器: max/validator
- 切面编程 max/aop
- API文档生成 max/swagger
